درگاه پرداخت رایان پی برای افزونه ووکامرس وردپرس
برای درخواست پذیرندگی و دریافت اطلاعات پذیرندگی (مرچنت) از لینک زیر اقدام نمایید :
https://rayanpay.com/

== Description ==

with this plugin, you can easily add rayanpay.com online payment gateway to woocommerce checkout system
با این پلاگین میتونید درگاه پرداخت رایان پی رو به سیستم تسویه حساب و پرداخت ووکامرس اضافه کنید


== Installation ==
 
just enable payment method and enter rayanpay.com merchant id in woocommerce checkout settings
راه اندازی و پیکربندی افزونه نیاز به تنظیمات خاصی نیست. فقط کافیه بعد از نصب و فعالسازی افزونه ، کد پذیرندگی خودتون رو در تنظیمات افزونه وارد کنید
برای ورود به صفحه تنظیمات افزونه از منوی ووکامرس ، زیرمنوی تنظیمات ، به سربرگ تسویه حساب رفته و روی اسم درگاه رایان پی کلیک کنید
