== Description ==

**VandaPardakht.com Payment Method for WooCommerce**
with this plugin, you can easily add vandapardakht.com gateway to woocommerce checkout system
با این پلاگین میتونید درگاه پرداخت وندا‌ پرداخت رو به سیستم تسویه حساب و پرداخت ووکامرس اضافه کنید

= مزایا =
 * سازگار با ووکامرس 2.2.x و 2.3.x و سری 3.x
 * پنل تنظیمات ساده و کاربرپسند
 * سازگاری کامل با کلاس ها و متد ها و روند کاری ووکامرس
 * نمایش کد رهگیری و خطاهای درگاه پرداخت به کاربر و مدیر سایت
 * قابلیت پرداخت بر اساس واحد های پولی ریال ، تومان و یا هزار تومان
 * بدون کارمزد و کاملا رایگان

== Installation ==
 
just enter vandapardakht.com pin
راه اندازی و پیکربندی افزونه نیاز به تنظیمات خاصی نیست. فقط کافیه پین دریافتی از وندا پرداخت رو در تنظیمات افزونه وارد کنید
برای ورود به صفحه تنظیمات افزونه از منوی ووکامرس ، زیرمنوی تنظیمات ، به سربرگ تسویه حساب رفته و روی اسم درگاه وندا‌ پرداخت کلیک کنید
