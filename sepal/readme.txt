== Description ==

with this plugin, you can easily add Sepal gateway to woocommerce checkout system
با این پلاگین میتونید درگاه پرداخت سپال رو به سیستم تسویه حساب و پرداخت ووکامرس اضافه کنید


== Installation ==
 
just enter sepal API-Key
راه اندازی و پیکربندی افزونه نیاز به تنظیمات خاصی نیست. فقط کافیه کد وبسرویس درگاه خودتون رو در تنظیمات افزونه وارد کنید
برای ورود به صفحه تنظیمات افزونه از منوی ووکامرس ، زیرمنوی تنظیمات ، به سربرگ تسویه حساب رفته و روی اسم درگاه سپال کلیک کنید
