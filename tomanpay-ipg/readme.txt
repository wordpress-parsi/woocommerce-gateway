== Description ==

with this plugin, you can easily add toman ipg (toman.ir) payment method to woocommerce checkout system
با این پلاگین میتونید روش پرداخت از طریق درگاه آنلاین تومن رو به سیستم تسویه حساب و پرداخت ووکامرس اضافه کنید


== Installation ==
 
install and activate the plugin. then go to plugin settings page and enter shop code and api secret
راه اندازی و پیکربندی افزونه بسیار ساده . افزونه رو نصب و فعال کنید
پس از فعالسازی بصورت خودکار به صفحه تنظیمات پلاگین هدایت خواهید شد
در صفحه تنظیمات پلاگین ، اطلاعات خواسته شده رو وارد کنید
