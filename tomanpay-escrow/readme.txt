== Description ==

with this plugin, you can easily add toman escrow service (tomanpay.net) payment method to woocommerce checkout system
با این پلاگین میتونید سیستم معاملاتی درگاه پرداخت امن تومن رو به سیستم تسویه حساب و پرداخت ووکامرس اضافه کنید
در پرداخت امن تومن، مبلغ پرداختی نزد تومن به امانت باقی می‌ماند و پس از انجام معامله در صورت تایید خریدار، تسویه با فروشنده صورت خواهد گرفت


== Installation ==
 
install and activate the plugin. then go to plugin settings page and enter shop code and api secret
راه اندازی و پیکربندی افزونه بسیار ساده . افزونه رو نصب و فعال کنید
پس از فعالسازی بصورت خودکار به صفحه تنظیمات پلاگین هدایت خواهید شد
در صفحه تنظیمات پلاگین ، شناسه فروشگاه و کد فعالسازی رو وارد کنید
